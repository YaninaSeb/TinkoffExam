Ограничение времени
1 секунда

Ограничение памяти
256 МБ

Василий неплохо отдохнул на новогодних каникулах, длящихся n дней. 
В i-й день праздников он занимался активностью под номером a. Василий 
называет отрезок [l;r] (1≤l≤r≤n) интересным, если с l-го по r-й день 
включительно он занимался хотя бы k различными активностями.
Помогите Василию понять, насколько прекрасно он провел каникулы — найдите 
количество интересных отрезков. 

Формат входных данных
Первая строка содержит числа n (2≤n≤2×100000) — длительность каникул.
Вторая строка содержит числа а1, а2, ..., aN (0≤a≤1000000000), записанные 
через пробел — номера активностей, которыми занимался Василий.
Третья строка содержит число k (1≤k≤n) — необходимое количество различных 
активностей, находящихся на отрезке, чтобы отрезок можно было считать интересным.

Формат выходных данных
Выведите одно число — количество интересных отрезков.

Замечание
В первом примере интересными отрезками являются отрезки [1,2], [1,3] и [2,3].
Во втором примере каждый отрезок является интересным. 
В третьем примере интересными отрезками являются отрезки [1,4], [1,5], [2,5], 
[3,5].
